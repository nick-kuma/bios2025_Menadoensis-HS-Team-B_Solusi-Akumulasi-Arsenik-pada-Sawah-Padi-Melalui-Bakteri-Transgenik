# bios2025_Menadoensis-HS-Team-B_Solusi-Akumulasi-Arsenik-pada-Sawah-Padi-Melalui-Bakteri-Transgenik

Aplikasi berbasis Tkinter Python untuk menampilkan riset biologi.

## Keperluan
- Python 3.10+
- Pillow

## Cara Menjalankan
1. Pastikan sudah menginstal Python 3.10+ dan Pillow.  
   Jika belum punya Python, unduh dari https://www.python.org/downloads
   
2. Pastikan semua berkas berikut ada di satu folder:
    - bios2025.py
    - pages (sudah unzipped)
    - diagram.png
    - peta.jpg

3. Jalankan perintah berikut pada Command Prompt untuk menginstal Pillow
```
pip install pillow
```

4. Jalankan perintah berikut pada Command Prompt untuk menjalankan aplikasinya:
```
python bios2025.py
```
