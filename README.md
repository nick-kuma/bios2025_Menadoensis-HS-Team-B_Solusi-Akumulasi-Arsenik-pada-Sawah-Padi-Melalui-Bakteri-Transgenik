# bios2025_Menadoensis-HS-Team-B_Solusi-Akumulasi-Arsenik-pada-Sawah-Padi-Melalui-Bakteri-Transgenik

Aplikasi berbasis Tkinter Python untuk menampilkan riset biologi.

## Keperluan
- Python 3.10+
- Pillow

## Cara menjalankan
1. Pastikan semua berkas berikut ada di satu folder:
    - bios2025.py
    - pages (sudah unzipped)
    - diagram.png
    - peta.jpg

2. Jalankan perintah berikut pada Command Prompt:
```
pip install -r requirements.txt
python bios2025.py
```
