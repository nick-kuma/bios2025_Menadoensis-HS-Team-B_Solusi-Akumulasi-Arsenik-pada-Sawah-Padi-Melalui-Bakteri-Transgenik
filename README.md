# bios2025_Menadoensis-HS-Team-B_Solusi-Akumulasi-Arsenik-pada-Sawah-Padi-Melalui-Bakteri-Transgenik

Aplikasi berbasis Tkinter Python untuk menampilkan riset biologi.

## Keperluan
- Python 3.10+
- Pillow

## Cara Menjalankan dengan File .exe
1. Unduh file dan ekstrak file .zip dari bagian Releases di Github.
2. Pastikan semua berkas berada dalam satu folder:
   - bios2025.exe
   - pages
   - diagram.png
   - peta.jpg
3. Jalankan aplikasi dengan klik dua kali bios2025.exe

## Cara Menjalankan dengan Python
1. Pastikan sudah menginstal Python 3.10+ dan Pillow.  
   Jika belum punya Python, unduh dari https://www.python.org/downloads <br>
   Saat instalasi, centang “Add Python to PATH”.
   
3. Pastikan semua berkas berikut ada di satu folder:
    - bios2025.py
    - pages (sudah unzipped)
    - diagram.png
    - peta.jpg

4. Buka Command Prompt dan jalankan berikut untuk menginstal Pillow:
```
pip install pillow
```

4. Jalankan aplikasi dengan perintah:
```
python bios2025.py
```
